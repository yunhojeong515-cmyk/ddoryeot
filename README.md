> 언제나 어디서나, 화면 위 어디든 탭 한 번으로 즉시 확대되는 노안 사용자를 위한 플로팅 돋보기

**또렷**은 노안(老眼)으로 스마트폰의 작은 글씨를 읽기 어려운 사용자를 위한 안드로이드 화면 돋보기 앱입니다. 어떤 앱을 사용하든 화면 위에 항상 떠 있는 작은 조준점을 탭하면, 그 주변을 실시간으로 확대해 보여줍니다.

---

## 주요 기능

- **플로팅 조준점** — 화면 가장자리에 항상 떠 있는 빨간 마커. 드래그로 자유롭게 이동하고, 손을 떼면 가까운 가장자리에 자동으로 붙습니다.
- **실시간 확대창** — 조준점을 탭하면 주변 화면을 약 25fps로 실시간 확대합니다. 마커를 옮기면 확대창도 함께 따라 움직입니다.
- **터치 투과 설계** — 확대창은 입력을 받지 않아, 아래 화면 조작을 방해 없이 그대로 사용할 수 있습니다.
- **배율 조절** — 2.0배 ~ 3.0배 사이에서 자유롭게 조절할 수 있으며, 확대창 위에서 두 손가락 핀치 줌으로도 조절됩니다.
- **비율 설정** — 1:1 · 4:3 · 16:9 중 원하는 확대창 비율을 선택할 수 있습니다.
- **회전 자동 대응** — 기기를 가로/세로로 돌려도 조준점과 확대창이 새 화면 크기에 맞게 자동으로 재배치됩니다.
- **드래그로 종료** — 대기 상태에서 조준점을 화면 하단으로 끌면 종료 버튼이 나타나, 거기에 놓으면 돋보기가 종료됩니다.
- **햅틱 피드백** — 주요 버튼과 슬라이더 조작 시 진동으로 조작감을 제공합니다.

## 기술 스택

- **언어**: Kotlin
- **최소 지원 버전**: Android 8.0 (API 26)
- **타깃 버전**: Android 14 (API 34)
- **화면 캡처**: MediaProjection API + VirtualDisplay + ImageReader
- **오버레이**: WindowManager (TYPE_APPLICATION_OVERLAY)

---

## 아키텍처

| 구성요소 | 역할 |
| --- | --- |
| `MainActivity` | 오버레이 · 화면 캡처 권한 요청 및 서비스 진입점 |
| `FloatingService` | 포그라운드 서비스. 조준점/확대창 오버레이 관리, 실시간 캡처·크롭·렌더링 담당 |
| `RatioActivity` | 배율·비율 설정 화면. 실행 중인 서비스에 변경 사항을 즉시 전달 |
| `Prefs` | SharedPreferences 기반 설정(배율, 비율) 저장소 |

### 핵심 설계 결정

- **조준점과 확대창을 별도의 오버레이 윈도우로 분리** — 확대창이 스스로를 비추는 미러링 현상을 방지합니다.
- **배율 하한(2.0배) 설정** — 저배율에서 확대창 자체가 캡처 영역에 포함되는 문제를 구조적으로 차단합니다.
- **화면 회전 시 VirtualDisplay를 재생성하지 않고 `resize()` 사용** — 최신 OS의 중복 생성 보안 정책 충돌을 회피합니다.
- **화면 잠금 시 조준점 유지 + 재동의 흐름** — 화면이 꺼지면 OS가 캡처 세션을 무효화하는데, 이때 조준점은 유지한 채 탭 한 번으로 다시 캡처 권한을 받아 복구합니다.
- **Android 14+ 전체화면 캡처 고정** — `MediaProjectionConfig.createConfigForDefaultDisplay()`로 앱 선택 없이 바로 전체화면 공유 동의창을 띄웁니다.

## 빌드 및 실행

1. 이 저장소를 클론합니다.
   ```bash
   git clone https://github.com/<your-username>/ddoryeot.git
   ```
2. Android Studio에서 프로젝트를 엽니다.
3. 안드로이드 기기(또는 에뮬레이터)를 연결하고 **Run** 버튼을 누릅니다.
4. 앱 실행 후 **다른 앱 위에 표시** 권한과 **화면 캡처** 동의를 허용하면 돋보기가 시작됩니다.

> 화면 캡처 권한은 Android 정책상 돋보기를 시작할 때마다 동의가 필요합니다. 이는 사용자 개인정보 보호를 위한 OS 차원의 정책으로, 앱이 임의로 생략할 수 없습니다.

## 사용 방법

1. **돋보기 시작** 버튼을 누릅니다.
2. 화면 가장자리에 나타난 빨간 조준점을 원하는 위치로 드래그합니다.
3. 조준점을 **탭**하면 그 주변이 확대되어 나타납니다.
4. 확대창 위에서 **두 손가락 핀치**로 배율을 조절할 수 있습니다.
5. 다시 조준점을 탭하면 확대창이 닫힙니다.
6. 종료하려면 대기 상태에서 조준점을 화면 하단의 종료 영역으로 끌어다 놓거나, 홈 화면의 **돋보기 끄기** 버튼을 누릅니다.
